# 📝 Python CLI Task Manager

## 📌 Overview

The **Python CLI Task Manager** is a simple yet efficient command-line application that allows users to manage their daily tasks. It demonstrates core programming concepts such as loops, functions, input handling, and basic data structures in Python.

This project is designed with clean code practices and proper error handling, making it beginner-friendly and interview-ready.

---

## 🚀 Features

* ✅ Add new tasks
* 📋 View all tasks with numbering
* ❌ Remove tasks by selecting their number
* 🔁 Continuous loop until user exits
* ⚠️ Handles invalid inputs gracefully (no crashes)

---

## 🛠️ Tech Stack

* **Language:** Python 3
* **Interface:** Command-Line (CLI)

## 💡 Usage Guide

Once the program starts, you will see a menu:

```
1. Add Task
2. View Tasks
3. Remove Task
4. Exit
```

* Select **1** to add a new task
* Select **2** to view all tasks
* Select **3** to remove a task by its number
* Select **4** to exit the program

---

## ⚠️ Error Handling

This application includes basic validation:

* Prevents adding empty tasks
* Handles non-numeric input when removing tasks
* Ensures task number exists before deletion
